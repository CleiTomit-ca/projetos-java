# Projetos Java

Projetos desenvolvidos durante meu aprendizado de Java Backend.

## Projetos

### Fundamentos
- **Caixa Eletrônico** — conta bancária com depósito, saque e extrato
- **Cadastro de Alunos** — sistema com média e situação dos alunos
- **Gerenciador de Despesas** — controle de gastos por categoria

### Intermediário
- **Sistema de Funcionários** — gerenciamento com persistência em arquivo
- **Sistema de Estoque** — controle de produtos

### Spring Boot + JPA
- **Sistema de Biblioteca JPA** — API REST com Spring Boot e PostgreSQL
- **Sistema de Barbearia** — API REST com agendamentos
- **Sistema de Delivery** — API REST com validações e tratamento de erros

## Tecnologias
Java 17 | Spring Boot | JPA | Hibernate | PostgreSQL | MySQL | Maven | REST API
